# Нейросеть для предсказания калорийности блюд

**Задача**
Создать и обучить нейросеть, которая будет предсказывать калорийность блюд.

**Цель**
Получить DL-модель, способную оценить калорийность блюда.

**Целевая метрика**
Модель будет считаться готовой при точности оценки MAE < 50 на тестовой выборке.

**Датасет**
Содержит фотографии блюд, описание ингредиентов, а также общую массу порции и калорийность.
